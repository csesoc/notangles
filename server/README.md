# Notangles server

## Installation

```bash
$ yarn install
```

## Running the server

```bash
# Development
$ yarn start

# Watch mode
$ yarn start:development

# Production mode
$ yarn build
$ yarn start:production
```

## Test

```bash
# Unit tests
$ yarn test

# End-to-end tests
$ yarn test:e2e

# Test coverage
$ yarn test:coverage
```

## Lint

```bash
# Lint
$ yarn lint

# Lint and fix
$ yarn lint:fix
```
